# sfe v0.2.2 Editing Polish Implementation Plan

> **For AI agents:** implement this plan with TDD. Keep the scope to v0.2.2 editing polish only; do not add larger Vim commands such as dd, yy, p, or gcc.

**Goal:** ship a small v0.2.2 patch that improves C editing ergonomics without changing the editor's broader feature set.

**Architecture:** keep the split between `sfe_core.py` for testable buffer behavior and `sfe.py` for curses input/rendering. Add focused buffer helpers for indentation/backspace behavior, keep search navigation state in `EditorApp`, and update packaging docs and version metadata.

**Tech stack:** Python standard library, curses UI, unittest, custom Debian package builder.

    buf.newline_with_indent(indent_width=4)
    assert buf.lines == ["    call(", "        ", "    )"]
    assert (buf.cursor_row, buf.cursor_col) == (1, 8)

def test_backspace_on_empty_indent_removes_one_indent_level():

    buf.cursor_col = 8
    assert buf.backspace_smart(indent_width=4)
